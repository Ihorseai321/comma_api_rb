# comma-api-rb

API Client for api.comma.ai

### Usage

```rb
me = CommaAPI.me()
puts("me:", me) # {"email":"makevoid@example.com","id":"12345....","points":2708,"prime":null,"regdate":1563123456,"superuser":false,"upload_video":false,"username":"antani12345"}
```

### Install

    bundle

### Run

    bundle exec rake
